14. Evidence Preservation

  1. Before moving an object, photograph the object in the location where you found it.
  1. Complete an evidence label. Place the label on the evidence bag before putting the object inside.
  1. Using gloves, place the object into the labelled evidence bag.
  1. Seal the bag completely.
  1. Hand the item to the Evidence/Exhibit Officer.

Certain objects must be sealed and stored in special ways. If you are unsure how to seal and store an object, consult a forensic specialist for advice before handling the object.
